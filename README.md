# Luxon - Premium Luxury Shopping Destination

<img src="https://res.cloudinary.com/drzkhibqf/image/upload/v1780810575/luxon_assets/lzunxlcjo2yish9vjwmq.png" width="80" alt="Luxon Logo">

**Live Demo:** [https://luxon-store.onrender.com/](https://luxon-store.onrender.com/)

Luxon is India's premier luxury shopping destination. It is a full-stack, responsive e-commerce platform designed to offer an elegant and seamless shopping experience for high-end products across various categories, including watches, bags, fashion, tech, and lifestyle.

---

## 🌟 Key Features

### For Users
*   **Premium User Interface:** A sleek, modern design with smooth animations, high-quality visuals, and a responsive layout built with Tailwind CSS.
*   **User Authentication:** Secure registration and login system.
*   **Product Catalog:** Browse a curated collection of luxury items with pagination, category filtering, and sorting options.
*   **Detailed Product Pages:** View high-resolution images, detailed descriptions, pricing, and stock availability.
*   **Wishlist:** Save favorite items for later with a dedicated wishlist page.
*   **Shopping Cart:** Add items to the cart, review the order, and proceed to checkout.
*   **Order Management:** Users can view their order history and profile details.

### For Administrators
*   **Secure Admin Dashboard:** A dedicated backend interface for store management.
*   **Inventory Management:** Add, edit, delete, and feature products.
*   **Category Management:** Organize products into categories and subcategories.
*   **Image Management:** Seamlessly upload and manage product images via Cloudinary integration.
*   **Order Tracking:** Monitor and manage customer orders.

---

## 🛠️ Tech Stack

*   **Backend:** Python, Flask
*   **Database:** SQLAlchemy (SQLite for local development / PostgreSQL for production)
*   **Frontend:** HTML5, Jinja2 Templates, Tailwind CSS (via CDN)
*   **Animations:** Motion One
*   **Media Storage:** Cloudinary
*   **Deployment:** Render

## 🚀 Getting Started (Local Development)

### Prerequisites
*   Python 3.8+
*   Git
*   A Cloudinary account (for image uploads)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/AyushBajaj7/Luxon.git
    cd Luxon
    ```

2.  **Create and activate a virtual environment:**
    ```bash
    # Windows
    python -m venv venv
    venv\Scripts\activate
    
    # macOS/Linux
    python3 -m venv venv
    source venv/bin/activate
    ```

3.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Set up environment variables:**
    Create a `.env` file in the root directory and add the following keys:
    ```env
    SECRET_KEY=your_super_secret_key_here
    CLOUDINARY_URL=cloudinary://<api_key>:<api_secret>@<cloud_name>
    ```

5.  **Initialize the Database:**
    ```bash
    flask db init
    flask db migrate -m "Initial migration"
    flask db upgrade
    ```

6.  **Seed the Database (Optional but recommended):**
    Populate the database with sample categories, products, and an admin user.
    ```bash
    python seed.py
    ```
    *Note: The default admin credentials generated by the seed script are typically `admin@luxon.com` / `admin123`.*

7.  **Run the application:**
    ```bash
    python run.py
    ```

8.  **Access the site:**
    Open your browser and navigate to `http://127.0.0.1:5000/`

## 📁 Project Structure

*   `/app`: The main application package containing blueprints, models, and forms.
    *   `/static`: CSS, JavaScript, and Image assets.
    *   `/templates`: Jinja2 HTML templates for the frontend and admin dashboard.
    *   `models.py`: SQLAlchemy database models.
    *   `routes.py`: Main frontend routes.
    *   `admin.py`: Admin dashboard routes and logic.
    *   `auth.py`: Authentication routes.
    *   `shop.py`: Cart, checkout, and wishlist routes.
*   `config.py`: Application configuration settings.
*   `run.py`: The entry point to run the application.
*   `seed.py`: Script to populate the database with initial data.
